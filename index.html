Revolutionizing Recruitment with CVNet

An intelligent CV filtering system that streamlines the hiring process through automated candidate ranking and API-driven integration.

Learn More View Architecture

Project Overview

Dual Interface System

Separate portals for job seekers and companies with tailored functionalities for each user type.

Smart CV Filtering

Advanced algorithms to filter and rank candidates based on GPA, skills, university, and other criteria.

API Integration

Companies receive custom APIs for seamless CV collection and integration with their existing systems.

Notification System

Real-time notifications for users about application status and for companies about new candidates.

How It Works

1

Dual Login System

Users can register/login as either individuals (job seekers) or companies (employers) with separate interfaces tailored to their needs.

2

Individual Profile Management

Job seekers can create and manage their profiles, upload CVs, add personal details, and track their applications.

3

Company Task Creation

Companies can create new recruitment tasks, each generating a unique API endpoint for CV collection.

4

Smart CV Filtering

Companies can filter received CVs using multiple criteria like GPA, university, skills, age, and experience level.

5

Seamless Integration

External company websites can integrate our CV collection system through provided APIs with popup authentication.

About CVNet

CVNet is an intelligent CV filtering system developed as a capstone project to revolutionize the recruitment process through automated candidate ranking and seamless API integration. With features like automated application tracking, secure cloud storage, and real-time dashboards, CVNet makes hiring faster, fairer, and more efficient.
